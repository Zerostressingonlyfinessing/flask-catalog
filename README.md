# Flask-магазин мебели для ванной

## Запуск

1. Установи зависимости:
```
pip install -r requirements.txt
```

2. Создай базу данных:
```
python create_db.py
```

3. Запусти приложение:
```
python app.py
```

Открой в браузере [http://localhost:5000](http://localhost:5000)
